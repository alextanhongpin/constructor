# constructor

Generate constructor for a given struct.



```
Usage of constructor:
  -exclude string
        the fields to exclude. Use comma to specify multiple fields.
  -in string
        the input file, defaults to the file with the go:generate comment
  -include string
        the fields to include. Use comma to specify multiple fields. * means all fields (default "*")
  -type string
        the type of the struct to generate. Use comma to specify multiple structs.
```
